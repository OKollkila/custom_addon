🚀 TOP RAM API Integration

Seamlessly integrate your Odoo Helpdesk with RAM Prime Care API. Automatically sync ticket stage changes with your external systems.

Odoo 18 Helpdesk API Integration Auto-Sync

✨ Key Features

📊 Workflow Level Management
Add configurable workflow levels (1-10) to your helpdesk tickets for better categorization and tracking.
🔄 Automatic API Synchronization
Instantly notify external systems when ticket stages change - no manual intervention required.
🔔 Real-Time Toast Notifications
Get instant visual feedback with color-coded notifications: green for success, orange for warnings, red for errors.
🛡️ Robust Error Handling
Built-in error handling ensures ticket updates proceed even if API calls fail.
📝 Comprehensive Logging
Track all API interactions with detailed logging for debugging and monitoring.

🎯 How It Works

  1. Set the Workflow Level on your helpdesk ticket (1-10)
  2. Change the ticket's Stage (e.g., New → In Progress → Done)
  3. The module automatically calls the RAM Prime Care API with:
    • Ticket ID
    • Workflow Level
    • New Stage Name
    • Update Timestamp
  4. Toast notification appears showing sync status (✅ success / ⚠️ warning / 🚫 error)
  5. External system receives real-time updates
📡 API Endpoint:
https://ramprimecare.com/HISAdmin/api/odooIntegration/updateRefundTask/{ticketId}/{workFlowLevel}/{status}/{updateTime}

📦 Installation

  1. Install Python dependencies: pip install requests
  2. Copy module to your Odoo addons directory
  3. Update Apps List in Odoo
  4. Install "TOP RAM API Integration"

🔧 Dependencies

  • Odoo Helpdesk module
  • Python requests library
